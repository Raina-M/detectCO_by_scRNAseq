# detectCO_by_scRNAseq
A pipeline for crossover identification based on scRNA-seq data from gametes, including recombination results analysis. Specifically, a project about the recombination of a holocentric plant - *R. breviuscula* was taken as the instance to delineate the pipeline.

Please cite the following publication when using our pipeline.
